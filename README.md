# Parallel Python
> A simple demonstration of a parallel program in python

## Example
```
$ python3 sequential.py

Using 1 CPU(s)
Parallel execution completed in 32.23907995223999 seconds.

$ python3 parallel.py
Using 4 CPU(s)
Parallel execution completed in 8.100968837738037 seconds.

```
